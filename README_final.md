# 📊 Classificação de Sentimento em Reclamações de Consumidores (CFPB)

## 1. 📌 Introdução
Projeto de NLP para análise de sentimento em reclamações financeiras usando dados da API CFPB.

## 2. 📥 Coleta de Dados
- Últimos 90 dias
- Paginação
- Controle de requisições

## 3. 🏷️ Rotulagem de Sentimento (Heurística)
Classificação baseada em frequência de palavras positivas e negativas.

⚠️ Limitação:
Pode não refletir o sentimento real.

## 4. 🧹 Pré-processamento
Limpeza, stopwords, lematização, tokenização e padding.

## 5. 🤖 Modelagem
Modelo LSTM com embeddings.

## 6. 📈 Avaliação
Métricas e matriz de confusão.

⚠️ Importante:
O modelo aprende a reproduzir a regra heurística.

## 7. 🔍 Análise das Dores
- Distribuição por produto
- Termos frequentes
- Temas
- Word clouds

## 8. ⚠️ Limitações
- Heurística
- Dados variáveis

## 9. 🚀 Melhorias
- BERT
- Dados anotados

## 10. 📌 Conclusão
Pipeline completo com insights relevantes.
